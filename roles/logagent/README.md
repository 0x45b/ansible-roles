# 变量

```text
logagent_download_url    # logagent 下载 url。默认 http://download.sensorsdata.cn/release/logagent/logagent_20191227.tgz
logagent_install_path    # loagent 安装路径。默认 /usr/local/logagent
logagent_path            # logagent 收集日志的路径。默认 /data/logs/salog
logagent_path_pattern    # logagent 匹配日志的 pattern。默认  sa.log.*
logagent_project         # loagent project。默认 production
logagent_server_host     # sensor 服务器的地址。需手动指定，必须项
logagent_server_port     # sensor 服务器的端口。默认8106
```
